# Лабораторная работа №3 (год 2)

## Библиотека Qt: QPainter

Разработана программа с использованием библиотеки Qt. В окне программы 
реализована возможность добавления геометрической фигуры посредством 
контекстного меню. Реализовано перемещение фигуры в рамках окна при 
перетаскивании ее за границу курсором мыши. При нажатии на границу фигуры 
правой кнопкой мыши выводится контекстное меню позволяющее повернуть ее или 
изменить размер. При наведении курсора на внутреннюю часть фигуры она
подсвечивается полупрозрачным цветом.

Для реализации фигуры использован класс QWidget и возможности классов 
QPainter и QPainterPath. 

По выполненной работе составлен отчет по ГОСТ 7.32, в котором отражены
изменения относительно лабораторной работы №2. Отчет содержит UML 2.0 диаграмму классов.
